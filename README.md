# Teerasak Komparw Profile

เว็บไซต์โปรไฟล์แบบเรียบง่ายสำหรับ `Teerasak Komparw` พร้อมฉาก 3D movement สร้างด้วย Vite และ Three.js

## Run locally

```powershell
node preview-server.mjs
```

แล้วเปิด:

```text
http://127.0.0.1:4173/
```

## Vite setup

เมื่อมี `npm` พร้อมใช้งาน สามารถรันแบบ Vite ได้ด้วย:

```powershell
npm install
npm run dev
```
